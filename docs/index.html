Port 8080

Node.js coordinator orchestrating 7 cron-scheduled AI agents. REST API + Redis pub/sub.

Port 3000

Next.js App Router control panel. Real-time P&L, risk gauge, agent status, ML training monitor.

🧠

Redis

Port 6379

Shared state bus. Agent findings, risk alerts, ML job logs — all flow through Redis.

🚀 Quick Start

# Clone and configure
git clone https://github.com/cheafi/trading-team.git
cd trading-team
cp .env.example .env    # fill in your keys

# Start full stack (dry_run mode by default)
docker compose up -d

# Open dashboard
open http://localhost:3000

# Backtest all strategies
./scripts/backtest-all.sh 20240101-20260101

# Retrain ML models
curl -X POST http://localhost:3001/api/ml/train
