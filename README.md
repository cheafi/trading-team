Check token in .env, restart agent-runner\n- No messages? Verify channel name (case-sensitive), bot perms\n- Commands not working? Enable MESSAGE CONTENT intent + re-invite\n\n## 📊 Strategies\n\n| # | Strategy | c | e | Style | Target WR% | Target DD% |\n|---|----------|------|-------|-------|-----------|-----------|\n| 1 | **A52** | 0.50 | -0.18 | Multi-factor momentum | 53% | <10% |\n| 2 | **OPT** | 0.65 | +0.05 | Ichimoku + SuperTrend | 48% | <12% |\n| 3 | **A51** | 0.35 | 0.00 | VWAP + Order Block scalper | 62% | <7% |\n| 4 | **A31** | 0.80 | -0.10 | Volatility squeeze breakout | 42% | <14% |\n\n**Parameters:**\n- `c` = Position sizing coefficient (0.0-1.0, higher = more aggressive)\n- `e` = Entry bias (-0.5 to +0.5, negative = short bias, positive = long bias)\n\n## 🏃 Backtesting\n\n```bash\n# Backtest all strategies for 2024\n./scripts/backtest-all.sh 20240101-20241231\n\n# Backtest specific strategy\ndocker compose run --rm freqtrade backtesting \\\n    --strategy A52Strategy \\\n    --timerange 20240101-20241231 \\\n    --timeframe 5m \\\n    --trading-mode futures\n```\n\n## 🔧 Services\n\n| Service | Port | Description |\n|---------|------|-------------|\n| Dashboard | 3000 | Next.js trading team control panel |\n| Agent API | 3001 | Agent coordinator REST API |\n| Freqtrade | 8080 | Trading engine API |\n| Redis | 6379 | Shared state store |\n\n## 📁 Project Structure\n\n```\ntrading-team/\n├── agents/                     #
